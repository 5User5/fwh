# 微信服务号对接指南

## 📋 准备工作

### 1. 注册微信公众平台账号
- 访问：https://mp.weixin.qq.com/
- 注册类型：服务号（需要企业资质）
- 完成认证（可选，但建议完成以获得更多接口权限）

### 2. 服务器准备
- 拥有公网IP的服务器
- 配置HTTPS（微信要求必须使用HTTPS）
- 确保80/443端口开放

---

## 🔧 微信后台配置

### 步骤1：获取公众号基本信息

1. 登录微信公众平台
2. 进入 **开发 > 基本配置**
3. 记录以下信息：
   - **AppID**：wxXXXXXXXXXXXXXXXX
   - **AppSecret**：xxxxxxxxxxxxxxxxxxxxxxxxxxxx
   - **Token**：自定义（任意字符串，需与代码配置一致）
   - **EncodingAESKey**：随机生成（可选加密模式时使用）

### 步骤2：配置服务器地址

1. 在 **开发 > 基本配置** 中找到 **服务器配置**
2. 点击 **修改配置**
3. 填写以下信息：
   - **URL**：`https://你的域名/wechat`
   - **Token**：与代码中配置的一致
   - **EncodingAESKey**：自动生成
   - **消息加密方式**：推荐选择 **明文模式**（开发阶段）

4. 点击 **提交**

> ⚠️ 注意：提交前必须确保服务器已启动并能正确响应验证请求

---

## 🛠️ 代码配置

### 修改配置文件

打开 `config.js` 文件，修改微信配置部分：

```javascript
wechat: {
  enabled: true,                    // 启用微信服务
  appId: 'wx你的AppID',              // 替换为你的AppID
  appSecret: '你的AppSecret',         // 替换为你的AppSecret
  token: '你的Token',                // 与微信后台配置一致
  encodingAESKey: '',                // 加密模式时填写
  encryptMode: false                 // false=明文模式, true=加密模式
}
```

### 配置说明

| 配置项 | 说明 | 必填 |
|-------|------|------|
| appId | 微信公众号AppID | ✅ 是 |
| appSecret | 微信公众号密钥 | ✅ 是 |
| token | 自定义验证Token | ✅ 是 |
| encodingAESKey | 加密密钥 | ❌ 否（加密模式需要） |
| encryptMode | 是否启用加密 | ❌ 否（默认false） |

## 🔌 微信接口说明

### 1. 消息接收接口

**地址**：`POST /wechat`

**功能**：接收微信服务器推送的消息

**支持的消息类型**：
- 文本消息
- 图片消息
- 语音消息
- 事件消息（关注、取消关注等）

### 2. 消息回复

系统会自动处理消息并回复，支持：
- 普通文本回复
- 图文消息回复
- 天气查询
- AI对话

### 3. AccessToken管理

系统会自动获取和刷新 AccessToken：
- 有效期：7200秒
- 自动缓存到内存
- 过期自动刷新

## 🧪 测试验证

### 步骤1：验证服务器配置

1. 在微信后台点击 **提交**
2. 如果配置正确，会提示 **配置成功**
3. 如果失败，检查：
   - 服务器是否正常运行
   - URL是否可访问（必须HTTPS）
   - Token是否一致

### 步骤2：测试消息发送

1. 使用微信关注你的服务号
2. 发送消息：`你好`
3. 应该收到回复：`你好！我是SOLO Auto Model助手，很高兴为您服务！`

### 步骤3：测试天气功能

1. 发送：`今天天气怎么样`
2. 回复：`请问您想查询哪个城市的天气？`
3. 发送：`北京`
4. 回复：显示北京的实时天气信息

## 🌐 部署建议

### 本地开发
```bash
# 使用ngrok进行内网穿透
ngrok http 8000
# 获得类似 https://xxxx.ngrok.io 的地址
# 在微信后台配置 URL: https://xxxx.ngrok.io/wechat
```

### 生产部署
1. 购买云服务器（阿里云、腾讯云等）
2. 配置域名和HTTPS证书（推荐使用Let's Encrypt免费证书）
3. 使用PM2管理Node.js进程
4. 配置防火墙开放443端口

## 📝 常见问题

### Q1：服务器配置提交失败？
- 检查服务器是否能正常响应
- 确保URL是HTTPS
- 检查Token是否一致
- 确保服务器时间与微信服务器时间同步

### Q2：收不到消息？
- 检查服务器日志
- 确认消息加密模式一致
- 检查微信后台是否已启用消息接收

### Q3：Access Token过期？
- 系统会自动刷新，无需手动处理
- 如果频繁报错，检查网络连接

## 📌 注意事项

1. **HTTPS必须**：微信要求所有接口必须使用HTTPS
2. **服务器IP备案**：国内服务器需要备案
3. **接口频率限制**：微信有接口调用频率限制，注意控制调用次数
4. **消息格式**：严格按照微信消息格式返回
