## 🚀 快速部署（推荐方案一）

### 1️⃣ 上传文件到服务器
上传以下文件到服务器的 `/home/admin/fwh/` 目录：
- `simple-server-music-card.js`（推荐）
- `config.js`（必须已有）

### 2️⃣ 在服务器上执行
```bash
cd /home/admin/fwh

# 停止旧服务
pm2 delete wechat-assistant

# 启动新服务
pm2 start simple-server-music-card.js --name "wechat-assistant"

# 查看日志（确认服务正常）
pm2 logs wechat-assistant --lines 50
```

### 3️⃣ 在微信公众号测试
发送以下消息：
- `点歌 小幸运`
- `播放 七里香`
- `你是谁`

## 💡 关于"付费歌曲"

### ❌ 无法实现的：
- 直接在微信公众号播放付费歌曲
- 获取付费音乐的真实播放链接（版权保护太严）

### ✅ 可以实现的：
- 搜索所有歌曲（包括付费歌曲）
- 返回音乐平台链接
- 用户点击链接后，在对应APP中播放
- 如果用户自己是VIP，就能听付费歌曲

```
用户：点歌 七里香
```
