# 阿里云服务器部署指南

## 📋 服务器信息

- **公网IP**：120.27.242.217
- **系统**：Ubuntu 24.04
- **配置**：2核2G

---

## 🚀 第一步：登录服务器

### 方法一：使用阿里云控制台（推荐新手）
1. 点击截图中的 **「远程连接」** 按钮
2. 选择 **「Workbench」** 或 **「VNC」**
3. 输入用户名密码登录

### 方法二：使用 SSH 工具
1. 下载 PuTTY（Windows）：https://www.putty.org/
2. 输入 Host：`120.27.242.217`
3. 端口：`22`
4. 连接后输入用户名密码

---

## 📦 第二步：安装 Node.js

登录后，依次执行以下命令：

```bash
# 1. 更新系统
sudo apt update -y
sudo apt upgrade -y

# 2. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# 3. 验证安装
node -v     # 应该显示 v18.x.x
npm -v      # 应该显示 x.x.x
```

---

## 📂 第三步：上传代码

### 方法一：使用 scp 命令（本地电脑执行）
在您的本地电脑打开命令行（CMD或PowerShell）：
```bash
# 进入项目目录
cd d:\Desktop\服务号

# 打包项目
tar -czf server-app.tar.gz .

# 上传到服务器
scp server-app.tar.gz root@120.27.242.217:/root/
```

### 方法二：使用 FTP 工具
1. 下载 FileZilla：https://filezilla-project.org/
2. 连接：
   - Host：120.27.242.217
   - 用户：root
   - 密码：您的服务器密码
   - 端口：22
3. 将整个项目文件夹上传到 /root/ 目录

---

## 🛠️ 第四步：在服务器上安装依赖

登录服务器后执行：

```bash
# 进入项目目录
cd /root/服务号

# 安装依赖
npm install

# 如果 npm install 有问题，试试 cnpm
sudo npm install -g cnpm --registry=https://registry.npmmirror.com
cnpm install
```

---

## ⚙️ 第五步：使用 PM2 管理服务

```bash
# 1. 安装 PM2
sudo npm install -g pm2

# 2. 启动服务
pm2 start simple-server.js --name "wechat-assistant"

# 3. 设置开机自启
pm2 save
pm2 startup

# 4. 查看服务状态
pm2 status
pm2 logs wechat-assistant
```

---

## 🌐 第六步：配置防火墙

开放需要的端口：
```bash
# 查看防火墙状态
sudo ufw status

# 开启防火墙
sudo ufw enable

# 开放端口
sudo ufw allow 22/tcp      # SSH
sudo ufw allow 80/tcp      # HTTP
sudo ufw allow 443/tcp     # HTTPS
sudo ufw allow 8000/tcp    # 应用端口（临时）

# 查看已开放端口
sudo ufw status
```

## 🎯 第七步：配置域名和 HTTPS

### 1. 购买域名
- 在阿里云或腾讯云购买一个域名
- 在域名管理中，添加 A 记录：
  - 主机记录：`@` 或 `www`
  - 记录值：`120.27.242.217`

### 2. 申请免费 SSL 证书
在阿里云控制台申请免费证书：
1. 进入 SSL 证书服务
2. 申请免费证书（DV SSL）
3. 下载证书

### 3. 配置 Nginx 反向代理
```bash
# 安装 Nginx
sudo apt install nginx -y

# 配置 Nginx
sudo nano /etc/nginx/sites-available/wechat
```

复制以下内容：
```nginx
server {
    listen 80;
    server_name 你的域名;
    
    location / {
        proxy_pass http://localhost:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
# 创建软链接
sudo ln -s /etc/nginx/sites-available/wechat /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

### 4. 配置 HTTPS（可选，但微信要求）
后续配置 Let's Encrypt 或使用阿里云免费证书。

## 📱 第八步：配置微信后台

1. 登录 https://mp.weixin.qq.com/
2. 进入 **开发 > 基本配置**
3. 填写服务器配置：
   - **URL**：`https://你的域名/wechat`
   - **Token**：`SOLOAutoModel2024`
   - **EncodingAESKey**：自动生成
   - **消息加密方式**：明文模式
4. 点击提交
