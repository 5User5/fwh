# SOLO Auto Model 微信服务号

## 🚀 快速启动

### 一键启动（推荐）

双击运行：**一键启动服务.bat**

该脚本会自动：
1. 安装依赖（如果需要）
2. 启动后端服务（端口 8000）
3. 启动前端服务（端口 3000）
4. 自动打开浏览器

### 访问地址

- **主页面**: http://localhost:8000
- **管理后台**: http://localhost:3000
- **管理员账号**: admin / admin123

## 🛑 停止服务

双击运行：**停止服务.bat**

## 📝 使用说明

1. **首次使用**：双击"一键启动服务.bat"
2. **正常使用**：直接双击即可自动启动前后端
3. **测试对话**：在页面中输入消息测试（试试问关于减肥的问题）
4. **管理后台**：登录后可查看对话记录和系统统计

## ⚙️ 环境要求

- Node.js 16+ 
- 下载地址: https://nodejs.org/

## 💡 功能特点

- ✅ 微信服务号消息对接
- ✅ AI 智能对话（支持减肥领域识别）
- ✅ 完整管理后台
- ✅ 系统数据统计
- ✅ 一键启动

## 🔧 技术栈

- 后端: Node.js + Express
- 前端: React + Ant Design
- 数据库: 内存数据库（开发模式）
