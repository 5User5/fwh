## 🌐 第二步：配置域名和 HTTPS

### 2.1 域名解析
✅ **已完成！** 您的域名 `gubei.asia` 已配置解析：
- 主机记录：`www`
- 记录类型：`A`
- 记录值：`120.27.242.217`

### 2.2 安装 Nginx
```bash
apt install nginx -y
systemctl start nginx
systemctl enable nginx
```

### 2.3 配置 Nginx 反向代理
创建配置文件：
```bash
nano /etc/nginx/sites-available/wechat
```

复制以下内容：
```nginx
server {
    listen 80;
    server_name gubei.asia www.gubei.asia;
    
    location / {
        proxy_pass http://localhost:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
# 创建软链接
ln -s /etc/nginx/sites-available/wechat /etc/nginx/sites-enabled/

# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx
```

### 2.4 配置 HTTPS（SSL 证书）

#### 方案一：使用 Let's Encrypt 免费证书
```bash
# 安装 Certbot
apt install certbot python3-certbot-nginx -y

# 获取证书
certbot --nginx -d gubei.asia -d www.gubei.asia
```

#### 方案二：使用阿里云 SSL 证书（推荐）
如果您已申请阿里云 SSL 证书，请按以下步骤配置：

1. 在阿里云 SSL 控制台下载证书（选择 Nginx 类型）
2. 将证书文件上传到服务器 `/etc/nginx/ssl/` 目录
3. 修改 Nginx 配置文件：
```nginx
server {
    listen 443 ssl;
    listen 80;
    server_name gubei.asia www.gubei.asia;
    
    # SSL 证书配置
    ssl_certificate /etc/nginx/ssl/gubei.asia.pem;
    ssl_certificate_key /etc/nginx/ssl/gubei.asia.key;
    ssl_session_timeout 5m;
    ssl_ciphers ECDHE-RSA-AES128-GCM-SHA256:ECDHE:ECDH:AES:HIGH:!NULL:!aNULL:!MD5:!ADH:!RC4;
    ssl_protocols TLSv1 TLSv1.1 TLSv1.2;
    ssl_prefer_server_ciphers on;
    
    location / {
        proxy_pass http://localhost:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```
