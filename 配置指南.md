# SOLO Auto Model 配置指南

## 📖 重要说明

**SOLO（TRAESOLO）是 TRAE IDE 内置的 AI 编程助手，它本身是免费的，但 SOLO 是集成在 IDE 中的功能，没有独立的公开 API 接口供外部调用。**

但是！我们可以通过**配置国内免费/低成本的第三方 API** 来让这个微信助手使用真实的 AI 模型！

---

## 🌟 推荐方案（国内免费/低成本）

### 方案一：使用 Ollama 本地运行模型（完全免费）

**优点：完全免费、数据隐私、无限制**

1. 下载安装 Ollama：https://ollama.com/
2. 启动后在终端运行：
   ```bash
   ollama run qwen2.5:7b
   # 或者
   ollama run deepseek-r1:1.5b
   ```
3. 配置 `config.js`：
   ```javascript
   apiUrl: 'http://localhost:11434/v1/chat/completions',
   apiKey: '任意字符',  // Ollama 不需要密钥
   model: 'qwen2.5:7b',
   useMock: false,
   ```

### 方案二：使用国内免费 API 服务

#### 1. 免费额度测试

**推荐平台：**
- **智谱 AI（免费额度）**：https://open.bigmodel.cn/
- **通义千问（免费额度）**：https://dashscope.console.aliyun.com/
- **DeepSeek（免费）**：https://platform.deepseek.com/

#### 2. 配置示例

**智谱 AI 配置：**
```javascript
apiUrl: 'https://open.bigmodel.cn/api/paas/v4/chat/completions',
apiKey: '你的API密钥',  // 在控制台获取
model: 'glm-4-flash',
useMock: false,
```

**DeepSeek 配置：**
```javascript
apiUrl: 'https://api.deepseek.com/v1/chat/completions',
apiKey: '你的API密钥',
model: 'deepseek-chat',
useMock: false,
```

## 🔧 快速配置步骤

### 1. 获取免费 API 密钥

**推荐先试用 DeepSeek（新用户有免费额度）：**
- 访问：https://platform.deepseek.com/
- 注册账号 → 获取 API Key

### 2. 修改配置文件

打开 `config.js`，找到：

```javascript
soloAutoModel: {
  // 修改以下内容
  apiUrl: 'https://api.deepseek.com/v1/chat/completions',
  apiKey: 'sk-xxxxx...你的真实API密钥',  // ← 换成你的
  model: 'deepseek-chat',
  useMock: false,  // ← 改为 false
}
```

### 3. 重启服务

重新运行 `一键启动服务.bat`
