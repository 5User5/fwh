# 🎵 QQ音乐点歌功能 - 快速部署指南

## ✨ 好消息！无需Cookie配置！

我为你创建了一个简化版，**直接使用QQ音乐公开API，不需要复杂的Cookie配置！**

---

## 🚀 快速部署（3步搞定）

### 第1步：上传新文件到服务器
在你的服务器上，上传以下文件到 `/home/admin/fwh/`：
- `simple-server-QQmusic.js`

### 第2步：停止旧服务，启动新服务
在服务器终端执行：
```bash
cd /home/admin/fwh

# 停止旧服务
pm2 delete wechat-assistant

# 启动新的音乐版服务
pm2 start simple-server-QQmusic.js --name "wechat-assistant"

# 查看日志，确认服务正常
pm2 logs wechat-assistant --lines 50
```

### 第3步：在微信公众号测试
发送这些消息试试：
- `点歌 小幸运`
- `播放 七里香`
- `点一首 晴天`

## 🔧 技术细节

### 使用的API
- QQ音乐搜索API：`https://c.y.qq.com/soso/fcgi-bin/client_search_cp`
- 无需认证，直接调用

- ✅ 无需Cookie配置
- ✅ 搜索返回前3首歌曲
- ✅ 每首歌带QQ音乐链接
- ✅ AI对话功能保留
